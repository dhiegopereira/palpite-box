## Start with docker-compose.yml
    $ sudo docker-compose up -d  
## Show container docker
    $ sudo docker ps
## Stop container docker
    $ sudo docker stop <CONTAINER ID>
## Remove container docker
    $ sudo docker rm <CONTAINER ID>
